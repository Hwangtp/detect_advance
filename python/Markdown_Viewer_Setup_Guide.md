## 이 문서의 목적

이 문서는 다음 작업을 쉽게 진행할 수 있도록 만들어졌습니다.

- 브라우저에서 Markdown 파일을 바로 열기
- 필요한 브라우저 확장 설치하기
- `.md` 파일을 보기 좋은 형태로 읽기
- 설명 자료와 실행 파일을 구분해 활용하기

## 준비물

- 웹브라우저 설치 완료
- 인터넷 연결 가능
- `.md` 파일 준비 완료

## 1. 웹브라우저 실행

1. 웹브라우저를 실행합니다.
2. 주소창에 `chrome.google.com/webstore` 또는 `chromewebstore.google.com`을 입력합니다.
3. 크롬 웹스토어에 접속합니다.

## 2. Markdown Viewer 확장 설치

브라우저에서 `.md` 파일을 보기 편하게 읽기 위해 Markdown Viewer 계열 확장을 설치합니다.

### 설치 순서

1. 크롬 웹스토어 검색창에 `Markdown Viewer`를 입력합니다.
2. 목록에서 Markdown 미리보기 기능을 제공하는 확장을 선택합니다.
3. `Chrome에 추가`를 클릭합니다.
4. 설치 확인 창이 뜨면 `확장 프로그램 추가`를 클릭합니다.

## 3. 확장 프로그램 확인

설치 후에는 아래처럼 확인합니다.

1. 브라우저 오른쪽 위의 퍼즐 모양 아이콘을 클릭합니다.
2. Markdown Viewer 확장이 보이는지 확인합니다.
3. 자주 사용할 경우 고정 버튼을 눌러 상단에 표시합니다.

## 4. Markdown 파일 열기

### 방법 1. 드래그 앤 드롭

1. 파일 탐색기에서 `.md` 파일을 찾습니다.
2. 파일을 마우스로 끌어 브라우저 창 안으로 놓습니다.
3. 브라우저에서 Markdown 문서가 바로 열리는지 확인합니다.

### 방법 2. 파일 열기

1. 브라우저를 엽니다.
2. `Ctrl + O`를 누릅니다.
3. 열고 싶은 `.md` 파일을 선택합니다.

## 5. 문서 활용 방법

`.md` 파일은 설명과 정리 내용을 읽기 위한 문서입니다.  
`.py` 파일은 직접 실행해 보면서 결과를 확인하기 위한 코드 파일입니다.

두 파일을 함께 보면 개념과 실행 결과를 연결해 이해하기가 쉬워집니다.

## 6. 자주 생기는 문제

### `.md` 파일이 일반 텍스트처럼만 보일 때

- Markdown Viewer 확장이 제대로 설치되어 있는지 확인합니다.
- 브라우저를 다시 열어 봅니다.
- 다른 Markdown Viewer 확장을 설치해 비교해 봅니다.

### 드래그 앤 드롭이 되지 않을 때

- 브라우저 창이 열려 있는지 확인합니다.
- 파일을 브라우저 안쪽 본문 영역에 놓아 봅니다.
- `Ctrl + O` 방식으로 열어 봅니다.

### 글자가 이상하게 보일 때

- 파일 인코딩이 UTF-8인지 확인합니다.
- 다른 `.md` 파일에서도 같은 현상이 있는지 확인합니다.
- 브라우저를 다시 실행한 뒤 다시 열어 봅니다.

## 7. 체크리스트

- 브라우저가 실행되어 있다
- Markdown Viewer 확장이 설치되어 있다
- `.md` 파일을 브라우저에서 열 수 있다
- `.md`와 `.py`의 역할 차이를 이해했다

## 한 문장 정리

브라우저에 Markdown Viewer를 설치하면 `.md` 파일을 문서처럼 읽을 수 있고, `.py` 파일과 함께 보면 학습 내용을 더 쉽게 이해할 수 있습니다.
