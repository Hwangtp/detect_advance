## 1. 09_IO_1 다음에 왜 이런 내용이 오는가

기본 입출력에서는 보통 아래 내용을 먼저 배웁니다.

- `input()`으로 사용자 입력받기
- `print()`로 결과 출력하기
- 파일 열기와 닫기
- 텍스트 파일 읽기와 쓰기

하지만 실제 프로그램에서는 여기서 한 단계 더 나아가야 합니다.

- 파일 경로를 더 안전하게 다루기
- 한글이 깨지지 않도록 인코딩 지정하기
- 구조화된 데이터를 JSON으로 저장하기
- 파일이 없을 때 오류 없이 처리하기

이 파일은 바로 그 다음 단계의 내용을 다룹니다.

---

## 2. pathlib로 파일 경로 다루기

파이썬에서는 `pathlib` 모듈을 이용해 파일 경로를 더 편리하게 다룰 수 있습니다.

```python
from pathlib import Path

base_path = Path(".")
text_file = base_path / "notes.txt"
```

`Path(".")`는 현재 폴더를 뜻합니다.  
`/` 연산자를 사용하면 경로를 자연스럽게 이어 붙일 수 있습니다.

`pathlib`를 사용하면 문자열을 직접 이어 붙이는 것보다 읽기 쉽고 실수를 줄이기 좋습니다.

---

## 3. 텍스트 파일을 UTF-8로 읽고 쓰기

텍스트 파일을 다룰 때는 `encoding="utf-8"`을 함께 쓰는 습관이 중요합니다.

```python
with open("notes.txt", "w", encoding="utf-8") as file:
    file.write("파이썬 파일 입출력\n")
```

UTF-8은 한글을 포함한 다양한 문자를 안전하게 저장할 수 있는 대표적인 인코딩 방식입니다.

파일을 읽을 때도 같은 인코딩을 사용합니다.

```python
with open("notes.txt", "r", encoding="utf-8") as file:
    content = file.read()
```

이렇게 하면 글자가 깨질 가능성을 줄일 수 있습니다.

---

## 4. 파일 전체 읽기와 줄 단위 읽기

파일을 읽는 방식은 상황에 따라 달라집니다.

### 전체 읽기

```python
content = file.read()
```

파일 내용을 한 번에 문자열로 읽습니다.

### 줄 단위 읽기

```python
for line in file:
    print(line.strip())
```

파일의 각 줄을 차례대로 처리할 수 있습니다.  
줄마다 따로 다루어야 할 때 유용합니다.

---

## 5. 파일 존재 여부 확인

파일을 열기 전에 먼저 존재 여부를 확인하면 더 안전하게 처리할 수 있습니다.

```python
from pathlib import Path

Path("notes.txt").exists()
```

결과는 `True` 또는 `False`입니다.  
이 방법은 파일이 있는지 확인한 뒤 읽기를 시도하고 싶을 때 유용합니다.

---

## 6. JSON 파일이란?

JSON은 데이터를 구조적으로 저장할 때 매우 자주 쓰이는 형식입니다.  
파이썬의 딕셔너리와 리스트 구조를 파일로 저장하기에 잘 어울립니다.

예를 들어 아래와 같은 딕셔너리가 있다고 해봅시다.

```python
student = {
    "name": "Kim",
    "age": 20,
    "major": "Computer Science",
    "skills": ["Python", "SQL", "HTML"]
}
```

이 자료를 JSON 파일로 저장하면 텍스트 파일보다 구조를 유지하면서 보관할 수 있습니다.

---

## 7. JSON 파일 저장하기

JSON 파일 저장에는 `json.dump()`를 사용합니다.

```python
import json

with open("student.json", "w", encoding="utf-8") as file:
    json.dump(student, file, ensure_ascii=False, indent=2)
```

여기서 중요한 옵션은 다음과 같습니다.

- `ensure_ascii=False`: 한글이 유니코드 문자로 깨지지 않도록 함
- `indent=2`: 보기 좋게 들여쓰기해서 저장함

이 옵션을 주면 JSON 파일이 더 읽기 쉬워집니다.

---

## 8. JSON 파일 읽기

저장한 JSON 파일은 `json.load()`로 다시 읽을 수 있습니다.

```python
with open("student.json", "r", encoding="utf-8") as file:
    loaded_student = json.load(file)
```

이렇게 읽은 결과는 다시 파이썬의 딕셔너리나 리스트로 돌아옵니다.

```python
print(loaded_student["name"])
print(loaded_student["skills"])
```

즉, JSON은 단순한 문자열이 아니라 구조를 가진 데이터를 파일로 저장하고 다시 복원하는 방법입니다.

---

## 9. 예외처리와 함께 파일 읽기

존재하지 않는 파일을 열면 오류가 발생합니다.

```python
with open("missing.txt", "r", encoding="utf-8") as file:
    print(file.read())
```

이 경우 `FileNotFoundError`가 발생할 수 있습니다.  
그래서 `try-except`로 감싸면 프로그램이 갑자기 종료되지 않고 오류를 처리할 수 있습니다.

```python
try:
    with open("missing.txt", "r", encoding="utf-8") as file:
        print(file.read())
except FileNotFoundError:
    print("파일을 찾을 수 없습니다.")
```

이 방식은 실제 프로그램에서 매우 자주 사용됩니다.

---

## 10. 폴더 안의 파일 찾기

`pathlib`를 사용하면 특정 확장자의 파일을 쉽게 찾을 수 있습니다.

```python
for path in Path(".").glob("*.txt"):
    print(path.name)
```

위 코드는 현재 폴더 안의 모든 `.txt` 파일 이름을 출력합니다.

이 기능은 많은 파일을 다룰 때 특히 편리합니다.

---

## 11. 파일 정보 확인하기

`Path` 객체를 사용하면 파일 이름, 확장자, 절대 경로 같은 정보도 쉽게 확인할 수 있습니다.

```python
print(text_file.name)
print(text_file.suffix)
print(text_file.resolve())
```

이런 정보는 파일 관리 프로그램이나 데이터 처리 프로그램에서 자주 활용됩니다.

---

## 12. 리스트와 파일 입출력 연결하기

파일에서 읽은 여러 줄을 리스트로 저장하는 패턴도 자주 사용됩니다.

```python
with open("todo.txt", "r", encoding="utf-8") as file:
    todos = [line.strip() for line in file]
```

이 코드는 파일의 각 줄을 읽어 공백과 줄바꿈을 제거한 뒤 리스트로 만듭니다.

즉, 파일 입출력은 자료구조와도 자연스럽게 연결됩니다.

---

## 13. 이 파일에서 익혀 두면 좋은 핵심

- 파일 경로는 `pathlib`로 다루면 편리하다
- 텍스트 파일에는 `encoding="utf-8"`을 붙이는 습관이 중요하다
- 구조화된 데이터는 JSON으로 저장하면 좋다
- 파일이 없을 수도 있으므로 예외처리가 필요하다
- 파일 내용을 읽어 리스트나 딕셔너리로 바꿔 활용할 수 있다

---

## 14. 정리

이 파일은 기본적인 텍스트 파일 입출력에서 한 단계 더 나아가, 실제로 많이 쓰는 실용적인 파일 처리 방법을 다룹니다.

- `pathlib`로 경로 다루기
- UTF-8 인코딩 사용하기
- JSON 파일 저장과 읽기
- 파일 존재 여부 확인하기
- 예외처리와 함께 안전하게 파일 열기

이 내용은 이후의 데이터 처리, 웹 프로그래밍, 데이터베이스, API 응답 저장 같은 주제로 자연스럽게 이어질 수 있는 기초가 됩니다.

## 코드

```python
import json
from pathlib import Path

base_path = Path(".")
text_file = base_path / "notes.txt"
json_file = base_path / "student.json"
print("텍스트 파일 경로:", text_file)
print("JSON 파일 경로:", json_file)

with open(text_file, "w", encoding="utf-8") as file:
    file.write("파이썬 파일 입출력\n")
    file.write("두 번째 줄입니다.\n")
    file.write("세 번째 줄입니다.\n")

with open(text_file, "r", encoding="utf-8") as file:
    content = file.read()
    print("\n파일 전체 내용:")
    print(content)

with open(text_file, "r", encoding="utf-8") as file:
    print("줄 단위 읽기:")
    for line in file:
        print(line.strip())

print("\nnotes.txt 존재 여부:", text_file.exists())
print("sample.txt 존재 여부:", Path("sample.txt").exists())

student = {
    "name": "Kim",
    "age": 20,
    "major": "Computer Science",
    "skills": ["Python", "SQL", "HTML"],
}

with open(json_file, "w", encoding="utf-8") as file:
    json.dump(student, file, ensure_ascii=False, indent=2)

with open(json_file, "r", encoding="utf-8") as file:
    loaded_student = json.load(file)
    print("\nJSON 파일에서 읽은 데이터:")
    print(loaded_student)

print("이름:", loaded_student["name"])
print("기술 목록:", loaded_student["skills"])

try:
    with open("missing.txt", "r", encoding="utf-8") as file:
        print(file.read())
except FileNotFoundError:
    print("\nmissing.txt 파일을 찾을 수 없습니다.")

print("\n현재 폴더의 txt 파일:")
for path in base_path.glob("*.txt"):
    print(path.name)

if text_file.exists():
    print("\n파일 이름:", text_file.name)
    print("확장자:", text_file.suffix)
    print("절대 경로:", text_file.resolve())

todo_list = ["자료구조 복습", "반복문 연습", "파일 입출력 정리"]
todo_file = Path("todo.txt")

with open(todo_file, "w", encoding="utf-8") as file:
    for item in todo_list:
        file.write(item + "\n")

with open(todo_file, "r", encoding="utf-8") as file:
    todos = [line.strip() for line in file]
    print("\n할 일 목록:", todos)

loaded_student["grade"] = 3

with open(json_file, "w", encoding="utf-8") as file:
    json.dump(loaded_student, file, ensure_ascii=False, indent=2)

print("\n수정된 JSON 데이터 저장 완료")
```
