# 03 데이터 업데이트 및 삭제

## 개요

이 문서는 기존 데이터를 수정하거나 삭제하는 SQL 구문을 다룹니다.

## UPDATE

`UPDATE`는 이미 저장된 값을 수정할 때 사용합니다.

```sql
UPDATE students SET age = 21 WHERE student_id = 1;
```

위 구문은 `student_id`가 1인 학생의 나이를 수정합니다.

여러 컬럼을 한 번에 수정할 수도 있습니다.

```sql
UPDATE students SET age = 23, grade = 'A' WHERE name = 'Bob';
```

## DELETE

`DELETE`는 조건에 맞는 데이터를 삭제할 때 사용합니다.

```sql
DELETE FROM students WHERE student_id = 3;
```

이 구문은 특정 학생 데이터를 삭제합니다.

## 주의할 점

- `WHERE` 절이 없으면 많은 데이터가 한 번에 수정되거나 삭제될 수 있습니다.
- 외래키 관계가 있는 경우 관련 데이터에 영향이 갈 수 있습니다.

## 변경 결과 확인

수정이나 삭제 후에는 `SELECT`로 결과를 다시 확인하는 습관이 중요합니다.

## 핵심 정리

- `UPDATE`: 기존 데이터 수정
- `DELETE`: 기존 데이터 삭제
- `WHERE`: 대상 범위 제한
