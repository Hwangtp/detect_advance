# 07 종합 정리

## 개요

이 문서는 지금까지 배운 데이터베이스 기초 내용을 한 번에 정리합니다.

## 앞에서 다룬 핵심 내용

### 1. 데이터베이스와 테이블 생성

- `CREATE DATABASE`
- `CREATE TABLE`
- 기본키와 외래키

### 2. 데이터 추가와 조회

- `INSERT`
- `SELECT`
- `WHERE`

### 3. 데이터 수정과 삭제

- `UPDATE`
- `DELETE`

### 4. 여러 테이블 연결

- `JOIN`
- `GROUP BY`

### 5. 중첩 조회

- 서브쿼리
- `EXISTS`

### 6. 성능과 재사용

- 인덱스
- 뷰

## 학습 흐름 정리

관계형 데이터베이스 학습은 보통 다음 순서로 이어집니다.

1. 구조 만들기
2. 데이터 넣기
3. 데이터 조회하기
4. 조건과 관계를 이용해 더 복잡하게 조회하기
5. 성능과 재사용 개념 이해하기

## 핵심 정리

- 테이블은 데이터를 구조적으로 저장합니다.
- 기본키와 외래키는 관계를 만듭니다.
- SELECT는 조회, INSERT는 추가, UPDATE는 수정, DELETE는 삭제입니다.
- JOIN은 여러 테이블의 관련 데이터를 함께 보여 줍니다.
- 서브쿼리는 쿼리 안에서 또 다른 쿼리를 사용합니다.
- 인덱스와 뷰는 성능과 재사용성에 관련된 개념입니다.
