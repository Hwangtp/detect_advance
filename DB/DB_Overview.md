# DB Overview

이 폴더는 MySQL 데이터베이스 16교시 강의 자료를 포함합니다. 총 8개의 SQL 스크립트 파일로 구성되어 있으며, 각 파일은 설명, 실습 쿼리, 미션 문제와 답을 포함합니다. 모든 쿼리에는 주석이 달려 있습니다.

## 파일 목록 및 설명

### 개요

- **00_DB_Overview.sql**: 데이터베이스 기초 개념과 용어 설명 (기본키, 외래키, 행, 열 등)

### DDL 및 DML

- **01.sql**: 데이터베이스 및 테이블 생성 (DDL)
- **02.sql**: 데이터 삽입 및 기본 조회 (DML)
- **03.sql**: 데이터 업데이트 및 삭제

### 고급 쿼리

- **04.sql**: 조인 (JOIN)
- **05.sql**: 서브쿼리 (Subquery)

### 최적화

- **06.sql**: 인덱스와 뷰

### 종합

- **07.sql**: 종합 미션

## 사용 방법

- MySQL 서버에 연결하여 각 파일을 순서대로 실행하세요.
- 설명을 읽고 실습 쿼리를 실행해 보세요.
- 미션 문제를 해결한 후 답을 확인하세요.

## 요구 사항

- MySQL 서버 설치
- MySQL 클라이언트 (Workbench 또는 CLI)
- 기본 SQL 지식

이 자료는 IT 강의를 위한 것으로, 1교시당 약 50분 분량입니다.
