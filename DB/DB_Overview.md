# DB Overview

이 폴더는 MySQL 기초 학습용 SQL 스크립트와 배포용 Markdown 문서로 구성되어 있습니다.

## 파일 구성

- `00_DB_Overview.sql`: 데이터베이스 기본 개념과 용어 정리
- `01.sql`: 데이터베이스와 테이블 생성
- `02.sql`: 데이터 삽입과 기본 조회
- `03.sql`: 데이터 수정과 삭제
- `04.sql`: 조인
- `05.sql`: 서브쿼리
- `06.sql`: 인덱스와 뷰
- `07.sql`: 종합 정리 및 연습

## 배포용 문서

- `00_DB_Overview.md`
- `01.md`
- `02.md`
- `03.md`
- `04.md`
- `05.md`
- `06.md`
- `07.md`

각 Markdown 파일은 학생 배포용 설명 문서입니다.  
개념과 핵심 예시 중심으로 정리되어 있습니다.

## SQL 스크립트 특징

SQL 파일에는 실행 순서에 맞는 설명 주석과 예상 결과 주석이 포함되어 있습니다.

예상 결과 주석은 다음과 같은 용도로 사용할 수 있습니다.

- 쿼리 실행 전 어떤 결과가 나올지 미리 파악하기
- 결과가 정상인지 비교하기
- 수업 자료를 만들 때 설명 보조용으로 사용하기

## 권장 사용 순서

1. `00_DB_Overview.md` 또는 `00_DB_Overview.sql`로 개념을 확인합니다.
2. `01.sql`부터 `06.sql`까지 순서대로 실행합니다.
3. 같은 번호의 `.md` 파일로 개념을 다시 정리합니다.
4. 마지막에 `07.sql`로 종합 연습을 진행합니다.
