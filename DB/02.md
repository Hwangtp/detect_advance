# 02 데이터 삽입 및 기본 조회

## 개요

이 문서는 DML을 사용해 데이터를 추가하고 기본적인 조회를 수행하는 방법을 다룹니다.

## DML이란?

DML(Data Manipulation Language)은 데이터를 조작하는 SQL 명령어입니다.

- `INSERT`
- `SELECT`
- `UPDATE`
- `DELETE`

## 데이터 삽입

### 학생 데이터 추가

`students` 테이블에 학생 정보를 추가합니다.

### 과목 데이터 추가

`subjects` 테이블에 과목 정보를 추가합니다.

### 성적 데이터 추가

`grades` 테이블에 학생별 과목 점수를 추가합니다.

이 과정에서 각 테이블은 서로 연결된 구조를 갖게 됩니다.

## 기본 조회

### 전체 조회

```sql
SELECT * FROM students;
```

테이블의 모든 컬럼과 모든 행을 조회합니다.

### 조건 조회

```sql
SELECT * FROM students WHERE age >= 20;
```

`WHERE` 절을 사용하면 조건에 맞는 데이터만 조회할 수 있습니다.

### 특정 컬럼 조회

```sql
SELECT name, grade FROM students;
```

필요한 컬럼만 선택해서 조회할 수 있습니다.

## 핵심 정리

- `INSERT`는 데이터를 추가합니다.
- `SELECT`는 데이터를 조회합니다.
- `WHERE`는 조건을 지정합니다.
- `*`는 모든 컬럼을 의미합니다.
