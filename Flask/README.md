# Flask 40교시 강의 교안

## Flask란 무엇인가?

**Flask**는 Python으로 웹 애플리케이션을 만드는 가벼운 웹 프레임워크입니다.

### Flask의 특징

- **Micro Framework**: 필수 기능만 포함하여 가볍고 빠름
- **유연성**: 필요한 기능을 자유롭게 추가할 수 있음
- **초보자 친화적**: Python 기초만 알면 쉽게 배울 수 있음
- **강력함**: 간단한 웹사이트부터 대규모 프로젝트까지 가능

### Flask를 왜 사용하는가?

1. **간단함**: 몇 줄의 코드로도 웹 서버 구축 가능
2. **빠른 학습**: Django 등 다른 프레임워크보다 진입장벽이 낮음
3. **확장성**: SQLAlchemy, 인증, 파일처리 등 라이브러리 자유롭게 선택
4. **디버깅**: 내장 개발 서버로 쉽게 테스트 가능
5. **커뮤니티**: 풍부한 문서와 예제, 많은 지원

### Flask의 동작 원리

```
1. 클라이언트가 웹 브라우저로 URL 요청
   ↓
2. Flask가 URL을 분석하고 라우트와 매칭
   ↓
3. 해당 함수(뷰)를 실행하여 데이터 처리
   ↓
4. HTML, JSON 등의 응답을 브라우저에 전송
   ↓
5. 브라우저가 응답을 화면에 표시
```

## 강의 구성

- 1일차 (8교시): Flask 기초, 라우팅, 정적 파일
- 2일차 (8교시): 템플릿, 폼 처리, 데이터베이스
- 3일차 (8교시): CRUD, 인증, 파일 업로드
- 4일차 (8교시): 게시판 통합, 고급 기능
- 5일차 (8교시): 최적화, 배포 준비

## 파일 목록

- **Flask_Overview.txt**: Flask 개요 및 설치 가이드
- **00_Hello_Flask.py**: 첫 번째 Flask 앱, 기본 라우팅
- **01_Routing.py**: 고급 라우팅, URL 매개변수
- **02_Static_Files.py**: 정적 파일(CSS, JS) 처리
- **03_Templates.py**: Jinja 템플릿 기초 (별도 설명 파일 참고)
- **04_Forms.py**: HTML 폼 처리
- **05_Database.py**: SQLite 데이터베이스 직접 연결
- **06_CRUD.py**: 데이터 생성, 읽기, 수정, 삭제
- **07_Authentication.py**: 사용자 인증 기초
- **08_File_Upload.py**: 파일 업로드 처리
- **09_Board.py**: 게시판 통합 (DB, 댓글)
- **Jinja_Template_Guide.txt**: Jinja2 템플릿 엔진 상세 설명
