# 08 File Upload

## 개요

이 문서는 Flask에서 사용자가 업로드한 파일을 저장하고, 목록 조회, 상세 보기, 다운로드, 삭제까지 처리하는 방법을 설명합니다.

## 업로드 폴더

파일 업로드 기능을 만들 때는 먼저 파일을 저장할 폴더가 필요합니다.

```python
UPLOAD_FOLDER = 'uploads'
app.config['UPLOAD_FOLDER'] = UPLOAD_FOLDER
```

필요하면 폴더를 자동으로 생성할 수도 있습니다.

## request.files

업로드된 파일은 `request.files`에서 읽습니다.

```python
file = request.files['file']
```

폼에서 파일 입력 요소의 이름과 맞아야 합니다.

## 파일 저장

선택한 파일은 서버 폴더에 저장할 수 있습니다.

```python
file.save(file_path)
```

## 파일 정보 저장

파일 자체만 저장하는 것이 아니라, 파일명, 크기, 형식, 설명 같은 정보를 데이터베이스에 함께 저장할 수 있습니다.

예를 들어

- 원본 파일명
- 서버 저장 파일명
- 파일 크기
- 파일 형식
- 업로드 시간

같은 정보를 관리할 수 있습니다.

## 파일 목록과 상세 보기

업로드된 파일 목록을 보여 주고, 특정 파일의 상세 정보를 확인하는 페이지를 만들 수 있습니다.

## 다운로드

Flask에서는 `send_file()`을 사용해 파일 다운로드를 제공할 수 있습니다.

```python
return send_file(file_path, as_attachment=True)
```

## 삭제

파일 삭제는 보통 두 작업이 함께 필요합니다.

- 실제 파일 삭제
- 데이터베이스 기록 삭제

둘 중 하나만 삭제하면 정보가 맞지 않을 수 있습니다.

## 통계

업로드 파일 수, 전체 용량, 파일 형식별 개수 같은 통계도 SQL과 함께 쉽게 만들 수 있습니다.

## 핵심 정리

- 업로드 파일은 `request.files`로 받습니다.
- 서버 폴더와 데이터베이스 기록을 함께 관리하는 것이 중요합니다.
- 업로드, 목록, 상세, 다운로드, 삭제가 기본 흐름입니다.
- 파일 기능도 결국 Flask 라우팅과 데이터베이스 처리의 응용입니다.
