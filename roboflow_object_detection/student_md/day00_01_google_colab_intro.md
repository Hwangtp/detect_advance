## 글로 이해하는 코랩

### 1. 코랩은 무엇일까?
구글 코랩은 웹브라우저 안에서 파이썬 코드를 작성하고 실행할 수 있는 온라인 실습 공간입니다. 프로그램을 많이 설치하지 않아도 되고, 설명과 코드를 한 화면에 같이 적을 수 있어서 수업용으로 매우 편리합니다.

### 2. 왜 수업에서 코랩을 사용할까?
학생마다 컴퓨터 환경이 다르면 설치 문제 때문에 수업 시간이 많이 줄어듭니다. 코랩은 대부분 같은 방식으로 실행되기 때문에 수업을 시작하기가 훨씬 쉽습니다. 즉, "설치 때문에 막히지 않게 해 주는 공통 실습장"이라고 생각하면 됩니다.

### 3. 코랩의 장점은 무엇일까?
1. 브라우저만 있으면 사용할 수 있습니다.
2. 코드와 설명을 함께 적을 수 있습니다.
3. 자동 저장과 공유가 편리합니다.
4. GPU를 쉽게 켤 수 있어 딥러닝 실습에 좋습니다.

### 4. 코랩은 어떻게 사용할까?
코드 셀을 만든 뒤 코드를 적고 `Shift + Enter`를 누르면 실행됩니다. 설명은 텍스트 셀에 적을 수 있어서 노트처럼 정리하기 좋습니다. 그래서 수업을 따라가면서 코드, 결과, 메모를 한 곳에 남기기 쉽습니다.

### 5. GPU는 무엇이고 왜 켤까?
GPU는 많은 계산을 빠르게 처리하는 장치입니다. 간단한 `sklearn` 실습은 CPU로도 충분하지만, CNN이나 딥러닝처럼 계산이 많아지는 단계에서는 GPU가 큰 도움이 됩니다.

### 6. 코랩에서 GPU는 어떻게 켤까?
상단 메뉴에서 `런타임 -> 런타임 유형 변경 -> 하드웨어 가속기 -> GPU`를 선택하면 됩니다. 바꾸고 나면 딥러닝 실습을 더 빠르게 진행할 수 있습니다.

### 7. 이번 수업에서는 어떻게 쓸까?
이번 강의에서는 코랩 전용 흐름으로 파일을 단순하게 맞춥니다. 그래서 데이터는 보통 `data/파일이름.csv`처럼 읽도록 구성합니다. 복잡한 경로 코드는 줄이고, 실습 자체에 더 집중할 수 있게 만드는 목적입니다.
