# Day 00-1 강사용 대본

## 수업 목표

이 시간은 실습을 시작하기 전에 학생들이 구글 코랩을 낯설지 않게 받아들이도록 만드는 시간입니다. 핵심은 "왜 코랩을 쓰는지", "왜 GPU가 필요한지", "앞으로 수업을 어떤 방식으로 따라오게 될지"를 편하게 이해시키는 것입니다.

## 수업 시작 멘트

"오늘은 본격적인 머신러닝 수업에 들어가기 전에, 우리가 앞으로 실습할 공간부터 먼저 익히겠습니다. 오늘 다루는 내용은 모델 자체보다도, 앞으로 수업을 막히지 않고 따라오기 위한 준비 시간입니다."

"여러분이 수업을 듣다 보면 제일 먼저 막히는 부분이 설치입니다. 그래서 우리는 설치 문제로 시간을 빼앗기지 않도록 구글 코랩을 사용합니다."

## 핵심 설명 흐름

"코랩은 인터넷 브라우저에서 파이썬 코드를 실행할 수 있는 온라인 실습장입니다. 따로 복잡한 설치를 많이 하지 않아도 되고, 코드와 설명을 한 화면에서 같이 볼 수 있어서 수업에 아주 잘 맞습니다."

"왜 수업에서 코랩을 쓰는가를 학생 눈높이에서 설명하면 간단합니다. 컴퓨터마다 환경이 달라서 설치가 안 맞으면 수업 전체가 흔들리기 때문입니다. 코랩은 대부분 같은 방식으로 실행되기 때문에, 강사 입장에서도 안정적이고 학생 입장에서도 진입 장벽이 낮습니다."

"이제 GPU 이야기를 해 보겠습니다. 간단한 sklearn 실습은 CPU로도 충분하지만, CNN이나 딥러닝 단계로 가면 계산량이 많아집니다. 이때 GPU는 계산을 더 빠르게 처리해 주는 장치입니다. 즉 딥러닝 실습의 대기 시간을 줄여 주는 도구라고 이해시키면 됩니다."

## 현장 멘트 예시

"여기서 학생들에게 꼭 말해 주세요. GPU를 켠다고 해서 더 똑똑해지는 것은 아닙니다. 다만 계산을 더 빠르게 해 주기 때문에, 같은 모델을 더 현실적인 시간 안에 돌릴 수 있게 됩니다."

"상단 메뉴에서 런타임, 그다음 런타임 유형 변경, 그다음 하드웨어 가속기에서 GPU를 고르면 됩니다. 이건 나중에 CNN이나 딥러닝 실습에서 다시 강조할 부분입니다."

"또 하나 중요한 점은, 이번 강의에서는 경로를 일부러 단순하게 맞췄다는 점입니다. 복잡한 경로 처리 대신 `data/파일이름.csv`처럼 읽게 해 두었기 때문에, 학생이 코드 자체에 더 집중할 수 있습니다."

## 마무리 멘트

"오늘은 코랩이라는 실습 공간을 익혔고, 왜 이 환경을 쓰는지, 왜 GPU가 필요한지까지 확인했습니다. 다음 시간부터는 이 공간 안에서 바로 머신러닝 데이터를 읽고 실제 예측으로 들어가겠습니다."
