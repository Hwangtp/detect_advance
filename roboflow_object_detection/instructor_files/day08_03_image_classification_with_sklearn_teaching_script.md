# Day 08-3 강사용 대본

## 수업 목표

이미지를 벡터로 펼쳐도 분류 문제로 다룰 수 있다는 점을 학생이 이해하고, 이미지 분류와 표 데이터 분류의 연결성을 느끼게 만드는 것이 목표입니다.

## 수업 시작 멘트

"오늘은 이미지를 가지고 실제 분류를 해 보겠습니다. 다만 CNN으로 바로 가지 않고, 먼저 sklearn 방식으로도 이미지 분류가 가능하다는 점을 확인하겠습니다."

## 설명 흐름

"입력 데이터 shape가 나오면, 이미지 한 장이 64개의 숫자 특징으로 펼쳐졌다는 뜻입니다. 이 부분에서 '이미지라도 결국 모델에 들어갈 때는 숫자 벡터가 될 수 있다'고 설명해 주세요."

"학습/테스트 데이터 개수를 보면, 이미지 문제도 여전히 train/test split 구조를 따른다는 점을 강조할 수 있습니다."

"테스트 정확도가 나오면 학생들은 '이미지도 이렇게 분류가 되는구나'를 직관적으로 느낍니다."

"혼동행렬이 나오면 어떤 숫자를 어떤 숫자로 헷갈리는지 읽어 주세요. 이 해석이 학생에게 매우 중요합니다."

"확률 출력에서는 모델이 단순히 답만 말하는 것이 아니라, 여러 숫자 중 어느 쪽 가능성이 큰지를 함께 보고 있다는 점을 짚어 주세요."

## 마무리 멘트

"오늘은 이미지도 결국 숫자 배열이라는 점, 그리고 그 배열을 가지고 분류 모델을 학습시킬 수 있다는 점을 확인했습니다. 이제 CNN이 왜 이미지에서 더 강력한지 연결해서 생각하게 하면 좋습니다."
