# Day 07-2 강사용 대본

## 수업 목표

학습한 모델을 저장하고 다시 불러오는 흐름을 실무 감각으로 연결해 설명하는 것이 핵심입니다.

## 수업 시작 멘트

"오늘은 모델을 잘 만드는 것에서 한 걸음 더 나아가, 만든 모델을 어떻게 보관하고 다시 쓰는지 보겠습니다. 현장에서는 이 단계가 아주 중요합니다."

## 설명 흐름

"모델을 한 번 학습했으면, 매번 처음부터 다시 학습시키지 않고 저장해 두는 것이 효율적입니다."

"출력에서 저장 전 시험 정확도가 먼저 나오면, 이 값이 현재 모델 상태를 보여 주는 기준점이라고 설명해 주세요."

"그 다음 저장 파일 경로가 나오죠. 이건 모델이 실제 파일로 만들어졌다는 뜻입니다. 학생들에게는 '학습 결과를 파일로 포장했다'고 비유하면 이해가 쉽습니다."

"이후 다시 불러온 모델 점수가 동일하게 나오면, 저장과 불러오기가 제대로 되었다는 증거입니다. 여기서 학생들이 '아, 학습 결과를 이어서 쓸 수 있구나'를 느끼면 됩니다."

## 현장 멘트 예시

"불러온 모델로 예측한 결과가 바로 나오는 장면이 중요합니다. 이 부분을 보면서 '훈련은 과거에 끝났지만, 예측은 지금도 계속할 수 있다'고 설명해 주세요."

## 마무리 멘트

"오늘은 모델을 학습하는 것에서 끝나지 않고, 저장하고 다시 쓰는 흐름까지 확인했습니다. 이 과정은 실제 서비스나 프로젝트에서 반드시 필요합니다."
