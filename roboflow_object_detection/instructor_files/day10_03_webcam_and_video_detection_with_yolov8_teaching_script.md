## 2. best.pt의 의미 설명

강사 멘트:

"여기서 출발점은 `best.pt`입니다. 이 파일은 학습이 끝난 뒤 얻은 최종 가중치 파일입니다. 즉 지금까지의 모든 준비가 이 파일 하나에 모여 있다고 생각하면 됩니다."

"따라서 웹캠 탐지든 영상 탐지든 핵심은 이 `best.pt`를 올바르게 불러오는 것입니다."

---

## 3. 웹캠 탐지 설명

강사 멘트:

"웹캠 탐지는 `source=0`처럼 카메라 번호를 넣어서 실행합니다. `show=True`를 주면 실시간으로 화면에 결과가 뜹니다."

"여기서 학생들에게 꼭 말해 주세요. 웹캠 탐지는 입력이 실시간이기 때문에 속도와 안정성이 중요합니다. 그래서 YOLO 계열 모델이 많이 쓰입니다."

---

## 4. 영상 탐지 설명

강사 멘트:

"영상 탐지는 입력이 mp4 같은 파일이라는 점만 다릅니다. 내부적으로는 같은 모델을 사용합니다. 대신 `save=True`를 주어 결과 영상을 저장하는 경우가 많습니다."

"즉 웹캠과 영상은 완전히 다른 프로젝트가 아니라, source만 다른 같은 객체탐지 프로젝트입니다."

---

## 5. conf 와 iou 옵션 설명

강사 멘트:

"여기서 `conf`는 confidence threshold입니다. 너무 자신 없는 예측을 화면에 그리지 않도록 자르는 기준입니다. `iou`는 겹치는 박스를 정리할 때 사용하는 기준입니다."

"이 두 값은 결과 화면의 느낌을 많이 바꿉니다. 박스가 너무 많이 나오면 conf를 높여 보고, 겹침 정리가 이상하면 iou를 조정해 보게 하면 학생들이 훨씬 실전 감각을 얻습니다."
