# Day 09-2 강사용 대본

## 수업 목표

YOLO 라벨 형식과 데이터 준비 흐름을 학생이 순서대로 이해하도록 돕는 것이 목표입니다.

## 수업 시작 멘트

"객체 탐지에서 학생들이 가장 많이 헷갈리는 부분이 바로 라벨 형식입니다. 오늘은 YOLO가 데이터를 어떤 형태로 읽는지 아주 분명하게 정리하겠습니다."

## 설명 흐름

"클래스 이름과 class id가 나오면, 문자 이름을 숫자 번호로 바꾸는 과정이라고 설명해 주세요. 모델은 보통 문자보다 숫자를 다루기 쉽기 때문입니다."

"YOLO 라벨 한 줄 예시가 나오면, 이 다섯 값이 각각 무엇을 뜻하는지 아주 천천히 짚어 주세요. class_id, x_center, y_center, width, height입니다."

"이미지별 라벨 개수와 split 분포를 보면서, 탐지 프로젝트가 단순히 사진만 모으는 것이 아니라 구조를 맞추는 일이라는 점을 강조합니다."

"data.yaml 정보가 나오면, 학생들에게 '모델은 이 파일을 보고 학습 데이터 위치와 클래스 이름을 이해한다'고 설명하면 좋습니다."

## 마무리 멘트

"오늘은 탐지 프로젝트를 시작하기 전에 반드시 이해해야 하는 라벨 형식과 준비 순서를 정리했습니다. 실제 학습 전에 이 구조가 머릿속에 있어야 이후가 편해집니다."
