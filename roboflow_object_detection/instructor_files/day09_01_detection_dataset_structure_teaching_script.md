# Day 09-1 강사용 대본

## 수업 목표

객체 탐지 데이터가 분류 데이터와 어떻게 다르고, 바운딩 박스 좌표가 어떤 의미를 가지는지 학생이 이해하게 만드는 것이 핵심입니다.

## 수업 시작 멘트

"이제부터는 객체 탐지로 넘어갑니다. 여기서 가장 먼저 해야 할 일은 모델보다 데이터 구조를 이해하는 것입니다."

## 설명 흐름

"출력에서 열 이름을 보면 이미지 이름, 클래스, 좌표, split 정보가 함께 들어 있습니다. 분류 데이터보다 훨씬 정보가 많다는 점을 학생들에게 먼저 느끼게 해 주세요."

"앞부분 5개 행을 볼 때는, 한 줄이 이미지 하나의 물체 정보라고 설명하면 됩니다. 즉 무엇이 있는지뿐 아니라 어디에 있는지도 같이 적혀 있습니다."

"정규화 좌표를 픽셀 좌표로 바꾸는 부분이 중요합니다. 이 장면에서 학생들이 '아, 숫자 0.15 같은 값이 그냥 추상적인 수가 아니라 실제 위치를 뜻하는구나'를 이해해야 합니다."

## 마무리 멘트

"오늘은 탐지 모델보다 먼저 탐지 데이터를 읽는 법을 배웠습니다. 객체 탐지에서는 데이터 구조를 이해하는 사람이 훨씬 강합니다."
