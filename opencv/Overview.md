# opencv Overview

## 과정 개요

**객체탐지** 과정은 OpenCV를 사용하여 이미지와 비디오에서 객체를 탐지하는 방법을 학습합니다.

- **총 교시**: 40교시 (1교시 = 50분, 1일 = 8교시)
- **난이도**: 초급 ~ 중급
- **필요 지식**: Python 기초
- **목표**: OpenCV를 활용한 객체 탐지 구현
- **상태**: ✅ **완료** (모든 파일 생성 및 검증 완료)

- 2교시: 이미지 읽기/쓰기/표시 ✅
- 3교시: 기본 이미지 처리 ✅
